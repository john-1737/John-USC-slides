\begin{frame}[t]
    \frametitle{1. Financial Market Returns}
    \section{Financial Market Returns}
    \framesubtitle{Total Returns}

    Definition of \textbf{total return}:
    $$\text{Return (\%)} = \frac{\text{Income}}{\text{Initial Price}} + \frac{\text{Change in Price}}{\text{Initial Price}} $$

    \vspace{1em}
    Notes:
    \begin{itemize}
        \item This is a \textit{backwards looking} formula.   
        \item Calculating total return requires three values: beginning and ending price and income.   Given total return and two of the three values, could calculate the third value by rearranging the equation.
        \item Expressed as a percentage of the initial price.
        \item Don't forget income!  Only include income that was earned during the holding period.
    \end{itemize}  
\end{frame}

    \framesubtitle{Measurement}

    Total Returns:
    
    $$
    \text{Return (\%)} = \frac{\text{Income}}{\text{Initial Price}} +
    \frac{\text{Change in Price}}{\text{Initial Price}} $$

    \vspace{2em}
    Alternative terminology:
    
    $$
    \text{Return (\%)} = \text{Income Return} + \text{Capital Gain/Loss}
    $$

\end{frame}

\begin{frame}[t]
    \frametitle{1. Financial Market Returns}
    \framesubtitle{Measurement}

    \textbf{Example}:  Coca-Cola (KO)
    \vspace{1em}
    
    Price:
        \begin{itemize}
            \item Jan 1, 2025:  \$61.87
            \item Dec 31, 2025: \$69.93
        \end{itemize}
	Dividends: 
        \begin{itemize}
            \item \$0.51 per quarter x 4 quarters = \$2.04
        \end{itemize}

	Total Return: 
         \begin{itemize}
             \item (\$2.04 + (\$69.93-61.87)) / \$61.87 = 16.4\%
         \end{itemize}

\end{frame}

\begin{frame}[t]
    \frametitle{1. Financial Market Returns}
    \framesubtitle{Measurement}

    \textbf{Example 2}:  10-year Treasury bond, 4.25\% coupon
    \vspace{1em}
    
    Price:
        \begin{itemize}
            \item Jan 1, 2025:  \$97.26
            \item Dec 31, 2025: \$101.00
        \end{itemize}
	Coupon Payments: 
        \begin{itemize}
            \item \$2.125 semi-annually x 2 = \$4.25
        \end{itemize}

	Total Return: 
         \begin{itemize}
             \item (\$4.25 + (\$101-97.26)) / \$97.26 = 8.2\%
         \end{itemize}

\end{frame}

\begin{frame}[t]
    \frametitle{2. Real v. Nominal Returns}
    \section{Real v Nominal Returns}

    \textbf{Definitions}

    \begin{itemize}
        \item \textbf{Nominal returns}: the percentage change in the value of an investment measured in current dollars.   Not adjusted for inflation.
        \item \textbf{Real returns}: Returns that are adjusted for inflation.   Real returns are the equivalent to the change in purchasing power.
        \begin{itemize}
            \item If prices increase, the real returns will be less than the nominal returns.   Think about an investment account that increases in value.   That's good!  But, if prices increase by the same amount, then you can only still buy the same amount of stuff with your larger investment account.
        \end{itemize}
    \end{itemize}

\end{frame}

\begin{frame}[t]
    \frametitle{3. History}
    \framesubtitle{Using historical returns}

    Often people use historical returns to predict expected returns.   This can be problematic.
    \begin{itemize}
        \item Financial returns often have ‘fat tails’
        \item Observations may not be independent
        \item The variance of returns may change over time
        \item Historical returns have no measure of value
    \end{itemize}
    
\end{frame}
